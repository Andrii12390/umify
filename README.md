# Umify

Umify is a web application for creating clean, structured UML diagrams in the browser.  
It focuses on speed, clarity, and a frictionless editing experience.

## Features

- Use case diagrams
- Class diagrams
- Package diagrams
- Node-based editor powered by React Flow
- Diagram export to PNG, JPG, SVG and PDF
- Light / dark theme switching
- Authentication via social providers
- Persistent storage for user diagrams

## Tech Stack

- Framework: Next.js 16, React 19, TypeScript
- Styling: Tailwind CSS, Shadcn UI
- Diagrams: React Flow
- Forms & Validation: React Hook Form, Zod
- Auth: NextAuth.js, social providers
- Database & ORM: PostgreSQL, Prisma,
- Testing: Vitest, Testing Library, Playwright
